How to use

You can use AJAX to call the TIENDA API and will receive a dic generated user in return.
If you are using jQuery, you can use the $.ajax() function in the code snippet below to get started..

See the Pen api bootcamp by Cristian López (@cristianl0pez) on CodePen.

Results

The application will provide you with a JSON, XML, CSV, or YAML object that you can parse and apply to your application.
You can specify the format you want the results in using the format parameter.
